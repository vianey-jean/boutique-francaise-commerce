# MANUEL UTILISATEUR

## Démarrage Rapide

### Première Connexion
1. **Inscription**: Créez votre compte avec email, mot de passe et informations personnelles
2. **Connexion**: Utilisez vos identifiants pour accéder au tableau de bord
3. **Configuration**: Le système est prêt à l'emploi, aucune configuration supplémentaire nécessaire

### Navigation Principale
- **Tableau de bord**: Vue d'ensemble et accès aux fonctionnalités
- **Tendances**: Analyses et graphiques de performance
- **Profil**: Gestion du compte utilisateur

## Gestion des Produits

### Ajouter un Produit
1. Allez dans **Tableau de bord > Inventaire**
2. Cliquez sur **"Ajouter un produit"**
3. Remplissez :
   - Description du produit
   - Prix d'achat
   - Quantité en stock
4. Optionnel : Ajoutez une image
5. Cliquez **"Enregistrer"**

### Modifier un Produit
1. Dans l'inventaire, cliquez sur l'icône crayon
2. Modifiez les informations nécessaires
3. Sauvegardez les changements

### Recherche de Produits
- Utilisez la barre de recherche (3+ caractères)
- Les produits en stock apparaissent en premier
- Filtrage automatique par stock disponible

## Gestion des Ventes

### Enregistrer une Vente
1. Allez dans **Tableau de bord > Ventes Produits**
2. Cliquez **"Ajouter une vente"**
3. Sélectionnez le produit (recherche intelligente)
4. Entrez :
   - Date de vente
   - Prix de vente
   - Quantité vendue
5. Le bénéfice se calcule automatiquement
6. Validez la vente

### Consulter l'Historique
- Filtrez par mois/année
- Exportez en PDF
- Consultez les statistiques détaillées

## Calculateur de Bénéfices

### Utilisation
1. Accédez à **Tableau de bord > Calcul Bénéfice**
2. Sélectionnez un produit
3. Configurez :
   - Prix d'achat
   - Taxe douanière (%)
   - TVA (défaut 20%)
   - Autres frais
   - Marge désirée (%)
4. Le système calcule :
   - Coût total
   - Prix recommandé
   - Bénéfice net
   - Taux de marge

### Sauvegarde
- Enregistrez vos calculs pour référence
- Consultez l'historique des calculs par produit

## Gestion des Prêts

### Prêts Familiaux
1. **Ajouter un prêt** :
   - Nom de la famille
   - Montant prêté
   - Date du prêt
2. **Enregistrer un remboursement** :
   - Sélectionnez le prêt
   - Entrez le montant payé
   - Le solde se met à jour automatiquement

### Prêts de Produits
1. **Créer un prêt produit** :
   - Sélectionnez le produit
   - Nom du client
   - Montant total
   - Avance versée
2. **Suivi des paiements** :
   - Montant restant calculé automatiquement
   - Statut : En cours / Payé

## Gestion des Dépenses

### Dépenses Mensuelles
1. Accédez à **Tableau de bord > Dépenses du Mois**
2. **Ajouter un mouvement** :
   - Type : Débit (sortie) ou Crédit (entrée)
   - Montant
   - Description
   - Date
3. Le solde se calcule en temps réel

### Dépenses Fixes
- Configurez vos charges récurrentes (loyer, assurances...)
- Elles s'appliquent automatiquement chaque mois

## Analyses et Tendances

### Graphiques Disponibles
- **Évolution des ventes** : Performance mensuelle
- **Répartition des bénéfices** : Par produit
- **Tendances annuelles** : Comparaisons
- **Stock rotation** : Produits les plus vendus

### Utilisation
1. Allez dans **Tendances**
2. Sélectionnez la période d'analyse
3. Consultez les métriques :
   - Chiffre d'affaires
   - Bénéfices totaux
   - Marge moyenne
   - Produits top ventes

## Résolution de Problèmes

### Problèmes Courants

**Produit non trouvé dans la recherche**
- Vérifiez l'orthographe
- Le produit a-t-il un stock > 0 ?
- Utilisez au moins 3 caractères

**Erreur de synchronisation**
- Vérifiez votre connexion internet
- Rechargez la page
- L'indicateur rouge indique une déconnexion

**Calculs incorrects**
- Vérifiez les paramètres de TVA et taxes
- Assurez-vous que les prix sont positifs
- Consultez l'historique des calculs

### Messages d'Erreur

**"Stock insuffisant"**
- La quantité demandée dépasse le stock
- Vérifiez l'inventaire du produit

**"Session expirée"**
- Reconnectez-vous
- Les sessions durent 24h

**"Données invalides"**
- Vérifiez les champs requis
- Respectez les formats demandés
